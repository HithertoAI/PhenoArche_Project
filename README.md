# PhenoArche_Project
The not-cloud. Synthetic geobio computation. No living cells. Just thermal gradients, salinity differentials, and asymmetric ion-selective membranes. Slower first, faster later. Built from first principles because the cloud is a dead end.

## Status

🧪 **Concept / Early R&D** — Not yet empirical. Hypotheses documented. Test fixtures in design. No living cells. No silicon. Just curiosity and a line in the sand.
